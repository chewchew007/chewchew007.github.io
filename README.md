# Template HBD GF By ITFeature

## Tech Stack

**React, TailwindCSS**

## การติดตั้ง

Install my-project with npm

```bash
  git clone
  cd your-project
```
Install node dependency

```bash
    npm install
```
Start Project 
```bash
    npm run dev
```
